Code example

All the code blocks that Asteri currently supports

meta.clr() - Clears the console.
IO.out="" - Prints a string. Does not support concatenation with variables.
IO.in:read - Records an input.
IO.out=IO.in - Prints the last recorded input.
win.goto="url" - Opens a specified url in a new opened tab in your default browser.
!!# this is a comment.
asteri!code-end - Prevents any code from running past this code.
asteri!code-end:true - An optional modification that prints a message when the code has finished running.
